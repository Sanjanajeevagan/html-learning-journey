# 🛍️ ShopEase - HTML Shopping Website

## 📖 About the Project

ShopEase is a simple shopping website built using **HTML5** as part of my **HTML Learning Journey**.

The goal of this project was to strengthen my HTML fundamentals by recreating a shopping website layout from a reference design and understanding how different HTML elements work together.

---

## ✨ Features

- 🏠 Home Section
- 🛒 Shop Section
- 📦 Product Table
- 📂 Categories Section
- ℹ️ About Us
- 📞 Contact Us
- 🔗 Navigation using Anchor Links
- 📋 Product Filters using Checkboxes

---

## 🛠️ Technologies Used

- HTML5

---

## 📚 HTML Concepts Practiced

- Semantic HTML (`header`, `nav`, `section`)
- Headings and Paragraphs
- Images
- Hyperlinks
- Lists (`ul`, `li`)
- Tables
- Forms
- Checkboxes
- Fieldset & Legend
- Internal Navigation using `id` and `href`

---

## 🎯 Learning Outcome

Through this project, I learned how to:

- Structure a complete webpage using semantic HTML.
- Organize content into meaningful sections.
- Create navigation links between sections.
- Display products using tables.
- Group related form elements using `fieldset` and `legend`.
- Build a simple multi-section shopping website using only HTML.
